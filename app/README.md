# mercado libre front-end test 
## Cliente y Render Server

## Stack Tecnológico: 
* React Server Side Rendering (SSR)
* SASS
* Css Grid - Flexbox
* NodeJS - Express

## Build

 para correr en modo producción

```
npm start
```

 para correr en modo development

```
npm run dev
```

## Test

 para correr test

```
npm test
```

 para correr tests con coverage

```
npm run coverage
```
