# ml-front-test

## Stack Tecnológico: 
* React Server Side Rendering (SSR)
* SASS
* Css Grid - Flexbox
* NodeJS - Express

## prerequisitos
Instalar NodeJS y todas las dependencias
```
npm install
```
## Build

### para correr en modo producción

```
npm run prod
```

### para correr en modo development
```
npm run dev
```
